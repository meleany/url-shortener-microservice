URL Shortener Microservice

FreeCodeCamp APIs & Microservices Project

The URL Shortener app accepts an URL as input and returns a JSON response containing the shortened version. To use it, enter the desired URL in the designated area above. The URL will be posted to the API endpoint:
POST [project_url]/api/shorturl/new

  • A valid URL returns a JSON response with a shortened version:
    {"original_url":"www.google.com","short_url":1}
  • An invalid URL returns a JSON response containing an error message:
    {"error":"invalid URL"}
  • Visiting the shortened URL will redirect to the original link.
  • You can use the function dns.lookup(host, cb) from the dns core module to check if the submitted URL points to a valid site.

Example of Creation:

POST [project_url]/api/shorturl/new - https://www.google.com

Example of Usage:

[this_project_url]/api/shorturl/l9AUnsxP

Expected Response (page redirects to):

https://www.freecodecamp.org
